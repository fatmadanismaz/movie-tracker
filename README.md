# 🎬 Movie Tracker

Modern ve kullanıcı dostu bir **Film Takip Uygulaması**. Bu proje React ve Bootstrap kullanılarak geliştirilmiştir. Kullanıcılar film ekleyebilir, düzenleyebilir, silebilir ve bilgiler tarayıcıda Local Storage sayesinde kalıcı olarak saklanır.

## 🚀 Özellikler

- ➕ Yeni film ekleme
- ✏️ Film bilgilerini düzenleme
- 🗑️ Film silme
- ⭐ Film puanı ekleme (0-10)
- 🎭 Film türü ekleme
- 📅 Çıkış yılı ekleme
- 💾 Local Storage ile verileri kaydetme
- 📱 Responsive ve modern kullanıcı arayüzü

## 🛠️ Kullanılan Teknolojiler

- React.js
- JavaScript (ES6+)
- Bootstrap 5
- HTML5
- CSS3
- Vite

## 📂 Proje Yapısı

```
src
│── Components
│   ├── MovieForm.jsx
│   └── MovieList.jsx
│── App.jsx
│── index.css
│── main.jsx
```

## ⚙️ Kurulum

Projeyi bilgisayarınızda çalıştırmak için:

```bash
git clone https://github.com/fatmadanismaz/movie-tracker.git
cd movie-tracker
npm install
npm run dev
```

## 📸 Uygulama

Film bilgileri eklenebilir, güncellenebilir ve silinebilir. Tüm veriler tarayıcı üzerinde Local Storage kullanılarak saklanır.
